# A Simple C++ Password Generator

## Features
- Selectable character sets (lower/upper/digits/symbols).
- Selectable password length between 8-24 characters.

## How to compile and run in GitHub Codespaces
1. From the GitHub repository.
2. Click the green **Code** button. 
3. Click the **Open With CodeSpaces** button.
4. From the GitHub CodeSpace click the **Run** button.

## How to build and run in the terminal
1. Install Git (if not already).
2. Download the code.
```bash
git clone https://github.com/ksd1100/cpp-creative-project
```
2. Compile (build) the program.
```bash
g++ main.cpp -o password_generator
```
3. Run the program.
```bash
./password_generator
```
